# HabitFlow – Health & Wellness Tracker

⚠️ Group Project – Hackathon (Masai School & NoBroker)

This project was developed as a team project during a hackathon conducted by Masai School in collaboration with NoBroker.

## My Contribution
- Developed responsive frontend components using React and Tailwind CSS
- Implemented JWT-based authentication and protected routes
- Integrated REST APIs for workout and nutrition features
- Integrated an AI-powered food analysis feature using the Gemini API to provide nutritional insights
- Implemented an AI chatbot using Gemini for basic health and food-related queries
- Collaborated with teammates under time constraints using Git and GitHub
